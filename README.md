## 🚀 What We Built

A full-stack production app serving live Nifty 50 analytics to retail traders — option chain, PCR, MaxPain, GEX, IV Greeks, FII/DII flows, and a 92-factor algorithmic signal engine.

 •🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React 18 + TypeScript + Vite |
| Mobile | Capacitor (Android) |
| Backend | Node.js on AWS EC2 Mumbai |
| Database | Firebase Firestore + MongoDB |
| Algo Engine | Cloudflare Workers (5 workers) |
| Auth | Firebase Authentication |
| Payments | Razorpay (live) |
| CI/CD | GitHub Actions (signed APK/AAB) |
| Data | Angel One SmartAPI (NSE licensed) |
